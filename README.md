# Simple Calculator App 🧮

A basic command-line calculator built using Python.  
It performs simple arithmetic operations:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## 🚀 How to Run

1. Make sure Python is installed (Python 3.x recommended).
2. Download or clone this repository.
3. Open terminal in the project folder.
4. Run:

   python calculator.py

## 📌 Features

- User-friendly CLI interface
- Handles division by zero errors
- Continuous calculations until user exits

## 🛠 Example

Enter first number: 10  
Enter operator (+, -, *, /): *  
Enter second number: 5  
Result: 50.0

## ❌ Exit

Type `exit` when prompted to stop the calculator.
